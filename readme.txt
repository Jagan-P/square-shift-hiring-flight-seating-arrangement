To run the application,
1. Clone the project into local machine that has Node.js installed
2. Run 'npm install' to resolve dependencies
3. Run 'node server.js' to start the express server
4. Open 'localhost:3000' to see the output

To change the inputs (rows-columns and number-of-passengers)
1. Open server.js
2. Change the value of 'rowsColumnsSeats' and 'numberOfPassengers'
3. Save the file
4. Stop and start the express server (or use nodemon)
